This virtual machine takes in a file of instructions in the form of integers and performs operations as set by the project's ISA.

**To compile the vm:**  
$ gcc vm.c  
**To run the vm:**  
$ ./a.out fileName.txt  
Where fileName.txt is the name of your input file.
